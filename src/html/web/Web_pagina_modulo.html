PROFESOR

Como profesor, tendrás acceso completo para gestionar los recursos educativos. Esto incluye las siguientes funcionalidades:

  • Añadir Recursos:

    Podrás subir nuevos materiales educativos como documentos, fotos, carpetas, etc., para que estén disponibles para los estudiantes.

  • Editar Recurso:

    Si necesitas actualizar algún recurso ya existente, podrás modificar su contenido, título o descripción para asegurar que la información esté al día.

  • Eliminar Recursos:

    Si un recurso ya no es relevante o debe ser retirado, podrás eliminarlo fácilmente.


ALUMNO

Como estudiante, podrás acceder y descargar los recursos que el profesor haya puesto a tu disposición. Las funcionalidades disponibles para ti son:

  • Descargar Recursos:

    Tendrás acceso a los materiales educativos subidos por los profesores, los cuales podrás descargar para su estudio o revisión posterior.
